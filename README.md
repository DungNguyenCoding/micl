# Machine Intelligence and Communications Lab (MICL)

> [Home Page](https://sites.google.com/site/junpyohong29/home)

## Research Area

* **Intelligent wireless communication systems**: we explore the application of machine learning algorithms to frequency resource management, transmission optimization, and network efficiency enhancement. 
* **Edge computing and communications**: We develop real-time data processing and low-latency communication technologies to support future services such as smart cities and autonomous vehicles. 
* **Cognitive networks and autonomous network management**: We design cognitive communication systems that can learn and adapt to network conditions, enabling autonomous networks capable of responding rapidly to changing environments. 
* **Deep learning-based signal processing**: We apply deep learning to signal processing and data transmission, enhancing communication efficiency and reliability in complex environments.
